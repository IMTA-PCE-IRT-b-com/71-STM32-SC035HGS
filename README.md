# 71-STM32-SC035HGS
Ce projet reprend toute la partie visuelle pour l'équipe 71.
Il contient les codes rédigés pour pouvoir utiliser le module caméra SC035HGS à l'aide un microcontrôleur STM32 (NUCLEO-H755ZI-Q).
Il contient également les informations concernant l'adaptateur interfaçant les deux composants : la documentation fournie plus loin dans ce document, et les schémas PCB (KiCAD) contenu dans ce dossier.
